# Meditação Cristã Autêntica

Um aplicativo simples e fiel à **Carta aos Bispos da Igreja Católica sobre alguns aspectos da meditação cristã** (15 de outubro de 1989).

## Sobre o projeto

Este MVP ajuda os fiéis a praticarem a oração cristã centrada em Cristo, na Sagrada Escritura e na doutrina da Igreja.

## Funcionalidades

- Leitura das principais seções da Carta
- Meditações diárias
- Diário Espiritual
- Interface simples e offline

## Como usar

Abra o arquivo `index.html` no navegador.
